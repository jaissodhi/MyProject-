# MyProject-

Number Guessing Game:
This is a simple Java game in which the computer generates a random number, and the user tries to guess it. The program provides feedback such as whether the guess is too high, too low, or correct, making it an interactive and fun project.

Online Examination System:
This is a Java-based application that allows users to take an online quiz or examination. It displays questions, accepts user answers, calculates the score, and shows the final result, demonstrating the use of arrays, loops, conditional statements, and user input handling.
